# OneDesk

A full-stack productivity and health management web application.

## Features

* User Authentication
* Task Management
* Notes Management
* Health Dashboard
* Food Tracking
* Water Tracking
* Nutrition Calculation
* MongoDB Database

## Tech Stack

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Express.js
* MongoDB
* JWT Authentication

Current Version: v2.0
